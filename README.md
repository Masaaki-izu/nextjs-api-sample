 # Next.js 学習サンプル（API連携）

Next.js / React を使ってAPIからデータを取得し、
画面に表示するサンプルアプリです。

## 機能
- APIデータ取得（fetch）
- 非同期処理（useEffect）
- 状態管理（useState）
- 一覧表示
- 再取得ボタン

## 技術
- Next.js
- React
- TypeScript
- Tailwind CSS

## 補足
Flutterでの非同期処理の経験を活かして、
Web側でもAPI連携の学習を行っています。

## URL
https://github.com/Masaaki-izu/nextjs-api-sample

## 学習目的
Next.jsの基本的なデータ取得・状態管理の理解を目的に作成しました。
